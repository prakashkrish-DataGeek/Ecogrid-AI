# EcoGrid AI: Global Spatial Intelligence Dashboard for Multi-Gigawatt AI Data Center Grid Alignment

An interactive, high-design geospatial intelligence platform analyzing proposed hyper-scale compute data center locations across the **United States, China, and India**. This engine intersects target megawatt (MW) capacities against localized grid constraints, transmission bottlenecks, and regional renewable energy accessibility.

The user interface aesthetics are directly modeled after the clean, content-first typography layout principles of **`about.google`**, merging production-grade GIS logic with premium corporate portfolio presentation standardizations.

---

## 🛠️ Core Business Case & Problem Landscape

As generative AI models and hyper-scale transformer workloads grow exponentially, data center development has shifted from standard real estate deployment to a complex, **resource-constrained energy sourcing problem**. 
* **The Sourcing Bottleneck:** Mega-clusters require hundreds of megawatts of continuous baseload capacity. Physical grid distribution systems are failing to scale at the pace of compute demands.
* **The Sustainability Mandate:** Enterprise customers demand strict Scope 2 greenhouse gas accounting. Selecting a location without understanding provincial grid carbon-intensity or open-access renewable legislation introduces significant compliance risks.

This platform bridges the strategic boardroom site-selection challenge with deterministic data engineering.

## 🏗️ Technical Architecture Architecture Blueprint

The script uses a decoupled compile-time injection pattern to construct a highly responsive single-file asset deployable at the edge via GitHub Pages:

```text
[Data Definition Layer]  --> Structural metadata arrays (Co-ordinates, Grid Strain, PUE Strategies)
           │
           ▼
[Folium GIS Engine]     --> Computes vector circle geometries (Radius ∝ MW Load)
           │                 Maps color-coded hexadecimal classes based on localized green profiles
           ▼
[Base64 Stream Compiler] --> Renders raw Leaflet JS parameters, converts to base64 string
           │                 (Eliminates cross-origin iframe security blocks)
           ▼
[Google UI Frame Injector]--> Injects binary stream into CSS/HTML grid scaffolding
           │
           ▼
[Output Pipeline Target] --> Produces self-contained `index.html` ready for CDN/GitHub edge hosting
